이 프로젝트는 TypeScript 코드를 간단히 추가하고 테스트 하는 템플릿 프로젝트 입니다.

## 시작하기

의존성 모듈 설치 및 개발서버 실행:

```bash
npm run install
npm run dev
# or
yarn install
yarn dev
```

## 비고

간단한 TypeScript 테스트 용 템플릿입니다.

- JavaScript 버전은 최신버전(ESNEXT)을 기본으로 합니다.
- 소스코드의 기본 폴더는 ./src/, 출력폴더는 ./dist/ 입니다.
